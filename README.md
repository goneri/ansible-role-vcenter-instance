Role Name
=========

Deploy a VCenter from the ISO on a ESXi host.

Requirements
------------

- an ESXI (tested with 6.5 or 6.7)
- the ISO of Vcenter (tested with VMware-VCSA-all-6.7.0-11726888.iso) 
- a Linux system
